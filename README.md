## 開発環境

### 開発環境起動

1. VS Code 起動
2. 左下のアイコンクリック
3. 「Remote-Containers: Reopen inContainer」クリック
4. しばらく待つ
   - 初回の場合コンテナ image の取得や作成が行われるため、それなりに時間がかかります
5. 起動したら開発可能
